基于netdisk-fast-download实现的在线解析服务


## 体验地址 https://lz.qaiu.top  |  
> 20250422 添加云盘监控接口
> 20241024 添加QQ音乐, 酷我音乐, 酷狗音乐, 网易云音乐分享链接的支持

网盘直链解析工具能把网盘分享下载链接转化为直链，已支持蓝奏云/蓝奏云优享/奶牛快传/移动云云空间/小飞机盘/亿方云/123云盘/Cloudreve/QQ音乐, 酷我音乐, 酷狗音乐, 网易云音乐等，支持加密分享。
开源地址: https://github.com/qaiu/netdisk-fast-download/ 使用Java技术栈, 和部署脚本, 本地化部署非常方便

## 1. 解析直链
- 操作步骤：输入分享链接和分享密码，点击“解析测试”。
- 界面演示
![image-1730273527154](https://bd2.qaiu.cn/blog/image-1730273527154.png)

## 2. 生成Markdown链接
- 操作步骤：输入分享链接和分享密码，点击“生成Markdown链接”。
- 功能说明：此功能旨在方便站长将下载链接嵌入到自己的网站中。
- 界面演示
![4c7f36280b6c7bd06c7bcf00ef62f91](https://bd2.qaiu.cn/blog/4c7f36280b6c7bd06c7bcf00ef62f91.jpg)


## 3. 生成二维码
- 操作步骤：输入分享链接和分享密码，点击“生成二维码”。
- 功能说明：此功能专为手机用户设计，通过生成二维码，用户可以使用手机浏览器扫描二维码直接下载文件，无需手动输入链接和密码，方便快捷。
- 界面演示
![67e86c7d9113a9e922ffa7e369d694d](https://bd2.qaiu.cn/blog/67e86c7d9113a9e922ffa7e369d694d.jpg)


## 3. 分享链接统计
- 操作步骤：输入分享链接和分享密码，点击“链接信息统计”。
- 功能说明: 通过点击提供的分享链接统计功能，您可以查看该分享文件的详细下载信息，包括但不限于下载次数。
- 链接有效性检查: 可以结合第三方状态检查服务进行邮件通知比如uptimerobot
